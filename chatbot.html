TITLE: Chatbot

CATEGORY: Data Visualization, Machine Learning

DATE: December 2016

TOOLS: Rive platform, Node.js, D3.js, Java Script

COLLABORATORS: None

DESCRIPTION:

'Chatbot Yummy' is a chatbot that gives you funny advices when it comes to food. This project is an exploration of my habits when buying food. I tracked all my grocery purchases made in the past 2 months and entered them into the database to make the visualization. The size of each ellipse is equal to the calorie amount of each grocery item per serving. Icons are designed by the Noun Project. For the chatbot I used the Rive platform.

URL: https://chatbot-yummy.herokuapp.com/
